Augmented Reality Convenience App

About this project

This is an Augmented Reality convenience application, which helps with basic activities for the day. The application allows user to check the current time and weather in Toronto and this information will be showed in the form of 3D models appearing above the image target. Another special feature for this application voice-controlled alarm system, which you can tell a basic command to make the alarm turn off.

Programming Language: C#
Tool: Unity, Vuforia
